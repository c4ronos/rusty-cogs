# Installation ✿
---
**1. Add the repository to your bot**
```text
repo add katana-cogs https://github.com/katanadotnet/katana-cogs
```
**2. Install the cog you want**
```text
cog install katana-cogs cogname
```
**3. Load the cog you installed**
```text
load cogname
```
